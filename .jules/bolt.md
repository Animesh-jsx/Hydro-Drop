## 2024-05-18 - Optimize scroll event listeners\n**Learning:** Unoptimized scroll event listeners that update state on every tick without throttling and without the `passive: true` flag can lead to unnecessary rapid re-renders and main thread blocking in React components.\n**Action:** Throttle scroll events using `requestAnimationFrame` with a ticking flag, apply `{ passive: true }` to the listener, and properly cleanup with `cancelAnimationFrame`.
